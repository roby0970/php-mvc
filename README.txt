Za pokretanje:

Instalirati XAMPP
Pokrenuti ga i u panelu pokrenuti Apache i MySQL
Odmah imate pristup MySQL Admin konzoli, odnosno PhpMyAdmin gdje mora importati SQL skripta koja je unutar repositorija
Podaci za pristup bazi su u kodu, i oni su defaultni (host = localhost, user = root) bez lozinke,
U browseru otvoriti url 'localhost'
Po defaultu bi trebalo otovoriti landing page od Apache communitya
Po defaultu folder koji se pokrece na tom URL-u je <XAMPP_installation_path>/htdocs
